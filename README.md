# Discord-Nuker

Requires .Net Core 3.1.0

![Discord Nuke](https://i.imgur.com/MLrEc5l.png)

## Features:
1. `Mass Create Channels & Ping`
2. `Delete All Channels`
3. `Delete All Roles`
4. `Delete All Emojis`
5. `Ban All Members`
6. `Kick All Members`
7. `Delete All Stickers`
8. `Give admin to specific user`
9. `Mass dm server members`
